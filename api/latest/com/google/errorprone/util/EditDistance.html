Class EditDistance

  • java.lang.Object
    • com.google.errorprone.util.EditDistance

  • public class EditDistance
extends Object
    A utility class for finding the edit distance between strings. The edit distance between two strings is the number of deletions, insertions, and substitutions required to transform the source to the target.
    Author:
    eaftan@google.com (Eddie Aftandilian)

Class ImportStatements

  • java.lang.Object
    • com.google.errorprone.apply.ImportStatements

  • public class ImportStatements
extends Object
    Represents a list of import statements. Supports adding and removing import statements and pretty printing the result as source code. Correctly sorts the imports according to Google Java Style Guide rules.
    Author:
    eaftan@google.com (Eddie Aftandilian)
